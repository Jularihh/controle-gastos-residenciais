# Controle de Gastos Residenciais

Projeto desenvolvido como teste para estágio.

Tecnologias utilizadas:
- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- React
- TypeScript
- Vite
- Axios

Como executar:

Backend:
- Abra a pasta Backend/ControleGastosAPI
- Execute:
dotnet run

Frontend:
- Abra a pasta Frontend/ControleGastosWeb
- Execute:
npm install
npm run dev

O sistema permite:
- Cadastro de pessoas
- Cadastro de transações
- Consulta de receitas, despesas e saldo
